# Rainfall Prediction using Machine Learning

This project predicts rainfall using machine learning classification techniques based on weather-related data.

## Project Overview

The goal of this project is to build a machine learning model that can predict whether rainfall is likely to occur based on weather features such as temperature, humidity, pressure, wind speed, cloud cover, and other environmental conditions.

## Technologies Used

* Python
* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Project Steps

1. Imported the required Python libraries
2. Loaded the rainfall dataset
3. Explored and cleaned the dataset
4. Checked missing values and data information
5. Performed data analysis and visualization
6. Separated input features and target values
7. Split the data into training and testing sets
8. Trained a machine learning classification model
9. Evaluated the model performance
10. Built a predictive system for rainfall prediction

## Output

The model predicts whether rainfall is likely:

* Rainfall
* No Rainfall

## What I Learned

This project helped me understand how machine learning can be applied to weather prediction. I learned about data preprocessing, feature selection, model training, evaluation, and building a simple prediction system using Python.
